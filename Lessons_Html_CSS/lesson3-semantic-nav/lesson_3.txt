Урок 3 — Семантика и навигация по якорям
Цель: использовать семантические теги и добавить навигацию по странице.
Время: 45 минут

План
Зачем семантика: SEO, доступность, структура.

Теги: header, nav, main, section, article, footer.

Навигация по якорям: id у секций и ссылки href="#about".

Мини-улучшения доступности: alt, единый порядок заголовков.

Проверка
Клики по меню плавно перемещают по секциям (без JS — просто якоря).

Код читаем, теги по смыслу.

Домашнее задание
Заполните секцию «Мои работы»: создайте 3 <article class="work"> c названием и 1–2 предложениями.

Подготовьте тексты для описаний карточек (будем переносить в карточки на Уроке 4).
