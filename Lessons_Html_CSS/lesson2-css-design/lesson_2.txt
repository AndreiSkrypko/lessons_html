Урок 2 — Подключаем CSS и создаём первый дизайн
Цель: подключить внешний CSS и оформить базовую типографику и шапку.
Время: 45 минут

План
Создать styles.css, подключить через <link rel="stylesheet" href="styles.css">.

Базовые стили: шрифт, цвета, отступы, фон, высота строки.

Оформить шапку и подзаголовки, сделать аккуратную картинку профиля.

Лёгкая декоративная палитра (спокойные цвета).

Ключевые моменты
Единый шрифт всей странице (например, системные).

Небольшой «контейнер» по ширине для читабельности.

Стили не должны «ломать» структуру (никакой магии — только базовые правила).

Проверка на уроке
Видно визуальное улучшение: фон, шапка с заливкой/градиентом, аккуратные отступы.

Картинка с рамкой и скруглением.

Домашнее задание
Примените стили к странице «Мой герой».

Подготовьте 3 изображения будущих работ (work1.jpg, work2.jpg, work3.jpg) в папку images/ (ширина 600–1200px).
